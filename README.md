## Project Overview
This project is a multi-page website for a South African streetwear brand called **Cozziano**. The website was designed to promote the brand, showcase products, provide business information, and allow customers to contact the company through enquiry forms.

Part 3 focused on adding JavaScript functionality, improving Search Engine Optimisation (SEO), implementing form validation, and enhancing the user experience.

---

## Website Goals
The goals of the website are:

- Promote the Cozziano streetwear brand.
- Display products in an attractive way.
- Allow customers to send enquiries.
- Provide contact information.
- Improve user interaction using JavaScript.
- Improve website visibility using SEO techniques.
